# Filter
A program that applies filters to BMPs

## Usage

### Compilation
$ make filter

### Filters

#### Greyscale
$ ./filter -g infile.bmp outfile.bmp

#### Sepia
$ ./filter -s infile.bmp outfile.bmp

#### Reflection
$ ./filter -r infile.bmp outfile.bmp

#### Blur
$ ./filter -b infile.bmp outfile.bmp
